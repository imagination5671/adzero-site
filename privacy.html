개인정보처리방침

애드제로(AdZero)는 개발자 서버로 이용자의 개인정보를 수집하지 않도록 설계된 앱입니다.

1. 수집하는 개인정보

애드제로 개발자는 앱을 통해 개인정보를 수집하지 않습니다.

  • 계정·회원가입·로그인이 없습니다.
  • 방문한 웹사이트 주소, 검색어, 브라우징 기록을 애드제로 개발자 서버로 전송하지 않습니다.
  • 광고 식별자, 분석(애널리틱스) 도구, 광고 SDK를 사용하지 않습니다.

2. 광고 차단이 동작하는 방식

애드제로는 Apple이 제공하는 Safari 콘텐츠 차단기(Content Blocker) 및 Safari 웹 확장 방식을 사용합니다. 차단 규칙은 기기 안에서 Safari가 직접 처리하며, 이 구조상 이용자가 어떤 페이지를 방문했는지 앱이나 개발자에게 전달되지 않습니다.

웹 확장의 요소 삭제(클리너) 선택 내역과 차단 통계는 이용자의 기기 안에만 저장되며 외부로 전송되지 않습니다.

3. 필터 업데이트

최신 차단 규칙을 받기 위해 앱이 공개 저장소(GitHub)에서 규칙 파일을 내려받을 수 있습니다. 이 과정은 일반적인 파일 다운로드 요청이며, 방문 기록·검색어 등 어떠한 이용 정보도 함께 전송되지 않습니다. 다운로드 과정에서 발생하는 표준 네트워크 기록(IP 주소 등)은 해당 인프라 제공자(GitHub)의 정책에 따라 처리되며, 개발자는 이에 접근하거나 이를 수집하지 않습니다.

4. DNS 보호 기능

이용자가 DNS 보호를 켜면, 기기의 DNS 질의가 이용자가 선택한 공개 DNS 서비스로 암호화(DoH)되어 전송됩니다. 이 기능은 이용자가 직접 켜기 전까지 동작하지 않으며, 개발자는 DNS 질의 내용을 볼 수 없습니다. 선택 가능한 DNS 서비스와 각 운영사의 개인정보처리방침은 다음과 같습니다.

  • AdGuard DNS — 개인정보처리방침
  • Cloudflare (1.1.1.2) — 개인정보처리방침
  • Quad9 — 개인정보처리방침

5. 공개 필터 데이터

애드제로의 앱과 Safari 확장 실행 코드는 애드제로가 직접 구현합니다. 광고 차단 규칙 데이터는 별도 라이선스의 공개 필터를 포함하며, YousList는 CC BY 4.0, EasyList 계열 파생 필터 데이터는 CC BY-SA 3.0+ 조건을 따릅니다. 원출처, 변경 사실, 파일별 라이선스는 공개 필터 라이선스 문서에서 확인할 수 있습니다.

6. 문의 시 처리되는 정보

이용자가 이메일로 문의하는 경우, 회신을 위해 보내주신 이메일 주소와 문의 내용만 처리하며 다른 목적으로 사용하지 않습니다. 문의 처리 완료 후 관련 법령상 보존 의무가 없는 한 지체 없이 파기합니다.

7. 외부 서비스와 정보 처리

애드제로 개발자는 이용자 정보를 판매하거나 광고 목적으로 제3자에게 제공하지 않습니다. 다만 필터 파일을 내려받을 때는 GitHub가, DNS 보호를 사용자가 켰을 때는 선택한 DNS 제공업체가 IP 주소·DNS 질의 등 서비스 운영에 필요한 정보를 각자의 개인정보처리방침에 따라 독립적으로 처리할 수 있습니다. 애드제로 개발자는 해당 제공업체의 운영 로그에 접근하지 않습니다.

8. 아동의 개인정보

애드제로는 연령과 무관하게 어떠한 개인정보도 수집하지 않으며, 아동으로부터 개인정보를 고의로 수집하지 않습니다.

9. 방침의 변경

기능 추가 등으로 본 방침이 변경되는 경우, 앱 업데이트 및 본 페이지를 통해 변경 사항을 고지합니다. 원격 기능이 새로 추가되어 정보 처리 방식이 달라지는 경우에는 변경 내용을 명확히 안내합니다.

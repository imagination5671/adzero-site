자주 묻는 질문을 먼저 확인해 보세요. 해결되지 않으면 메일로 문의해 주세요.

광고 차단이 시작되지 않아요

설치 후 한 번만 아래 단계를 따라 하면 돼요.

  • iPhone·iPad의 설정 앱을 엽니다.
  • iOS 18 이상에서는 설정 → 앱 → Safari → 확장 프로그램으로 이동합니다.
  • 애드제로 콘텐츠 차단기 5개와 Safari 도우미를 모두 켭니다.
  • Safari 도우미에서 '모든 웹사이트' 접근을 허용합니다.

개인정보 보호를 위해 앱이 대신 설정을 켤 수는 없어요. 켠 뒤에는 Safari에서 페이지를 새로고침해 주세요.

특정 사이트에서 광고가 남아 있어요

사이트들이 광고 방식을 수시로 바꾸기 때문에 일부 광고가 일시적으로 보일 수 있어요. 앱의 필터 탭 → 필터 업데이트로 최신 규칙을 받아 보시고, 그래도 남아 있으면 사이트 주소와 함께 메일로 알려주세요. 필터를 갱신해 드릴게요.

사이트가 이상하게 보이거나 기능이 안 돼요

앱의 필터 탭 → 화이트리스트에 해당 사이트를 추가하면 그 사이트에서만 차단이 꺼집니다. 같은 내용을 메일로 신고해 주시면 규칙을 수정해 다른 분들도 겪지 않게 반영할게요.

DNS 보호는 어떻게 켜나요?

앱의 설정 탭 → DNS 보호에서 원하는 서비스를 선택해 저장한 뒤, iOS 설정에서 애드제로 DNS를 켜면 적용돼요. 선택한 공개 DNS 서비스 정책에 따라 멀웨어·피싱·성인 도메인이 DNS 단계에서 차단될 수 있습니다.

YouTube 앱 광고도 차단되나요?

아니요. YouTube 앱의 동영상 광고 차단은 지원하지 않아요. 애드제로는 Safari 웹페이지의 광고 차단과 선택형 DNS 보호를 제공합니다.

문의하기

사이트 주소, 기기 종류(iPhone/iPad), 문제 화면 캡처를 함께 보내주시면 빠르게 도와드릴 수 있어요.
